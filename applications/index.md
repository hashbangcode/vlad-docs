<h1>Vlad Installed Applications</h1>

Vlad can install the following technologies:

- Apache2
- Adminer
- [Drupal Install Scripts](drupal.md)
- [Drush](drupal.md) (with the Site Audit & Registry Rebuild extensions)
- Git
- ImageMagick
- Mailcatcher
- Memcached
- [Munin](munin.md)
- [MySQL](mysql.md)
- Node.js
- [PHP](php.md)
- PimpMyLog
- Redis
- Ruby (via RVM)
- Sendmail
- Solr (version 4)
- [Varnish](varnish.md)
- [Xdebug](xdebug.md)
- [XHProf](xhprof.md)
